# APPQUEMAS


## Development server
Run `ionic serve` for a dev server. Navigate to `http://localhost:8100/`. The app will automatically reload if you change any of the source files.

## Build
Run `ionic capacitor build --prod --no-open` to build the project. The build artifacts will be stored in the `dist/` directory.
